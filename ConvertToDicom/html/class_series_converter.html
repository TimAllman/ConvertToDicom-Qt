Detailed Description

The SeriesConverter class.

Class to do the conversion of one series. It reads the image files, converts them to DICOM and writes them out as DICOM files.

◆ extractImageParameters()

ErrorCode SeriesConverter::extractImageParameters ( )

Tries to get as many metadata from the input image files as possible.

If the image is a DICOM series the metadata dictionary will be queried. Otherwise the data will likely be limited to dimensions and slice spacing. This must be called after readFiles().
